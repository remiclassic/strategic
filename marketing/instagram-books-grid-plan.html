Posting mix

  • Static Single 4∶5 image → fastest. Use AI poster or catalog cover.
  • Carousel Slide 1 = hook poster, slides 2–3 = one “what’s inside” bullet each (from whatsInside), last = CTA “Link in bio → strategicsloth.com/books”.
  • Reel 9∶16 text-on-yellow motion; end card → same CTA. Optional duplicate as static cover.

Linking: Warm traffic → https://www.strategicsloth.com/blog/<slug> when a post exists. Direct buyers → https://www.strategicsloth.com/books or bundle #bundles. Lemon checkout links stay in your shop / link-in-bio tool—not pasted in captions.
