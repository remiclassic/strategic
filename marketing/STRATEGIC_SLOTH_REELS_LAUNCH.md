## Visual system (quick reference)

| Element | Direction |
|--------|-----------|
| **Mood** | Calm power, honest, anti-hustle, premium minimal |
| **Type** | One headline weight + one body; high contrast, short lines |
| **Color** | Near-black / charcoal base, off-white text, one accent (sage or muted electric) |
| **Motion** | Slow, confident; snap cuts on beats; no chaotic zoom spam |
| **Sloth** | Sparingly — mark or silhouette, never cute-meme overload |

# 1. Mini visual style guide (series-wide)

### Fonts
- **Primary (headlines):** Geometric grotesk with personality — e.g. **Söhne**, **Neue Haas Grotesk**, **Satoshi**, **Outfit** (bold/black).  
- **Secondary (steps, captions on screen):** **IBM Plex Mono** or **JetBrains Mono** at small sizes for “system” feel — use only for labels (01/02/03), URL, “link in bio.”  
- **Rules:** Headline 72–120px equivalent at 1080 width; body on-screen max 42–56px; minimum readable mobile size ~36px for short words only.

### Color palette
| Role | Hex (suggested) | Use |
|------|-----------------|-----|
| Void black | `#0B0C0E` | Primary background |
| Charcoal | `#14161A` | Cards, gradients |
| Paper white | `#F2F0EA` | Primary text |
| Muted white | `#B8B5AD` | Secondary text |
| Accent (pick one) | `#6B8F71` sage OR `#5C7CFA` muted electric | Keywords, rules, CTA highlights |
| Danger (sparing) | `#8B3A3A` wine | Only for “NOT ME” / contradiction beats |

### Text treatment
- **Hooks:** ALL CAPS, tight line breaks, 2–6 words per line.  
- **Body beats:** Sentence case OK when lines run long.  
- **Max 3 lines** on screen at once; fourth line = new card.  
- **Safe area:** 64px inset minimum from edges; avoid top 15% if UI overlap matters.

### Motion treatment
- **Cuts:** On beat or end of phrase; 2–4 frames black flash acceptable once per reel max.  
- **Ken Burns:** Subtle 102–108% zoom over 3–4s on B-roll only.  
- **Text enter:** 8–12 frame slide-up with ease-out OR simple opacity (no bounce).  
- **Sound:** Optional low sub hit on pivot words; no generic “Instagram motivation” whoosh spam.

### Composition style
- **Asymmetric** with one strong vertical or horizontal rule (1px–2px, accent or white 20% opacity).  
- **Negative space** = premium; never fill frame with stickers.  
- **Edgy** via **contrast and wording**, not via clutter.

### Icon or sloth usage
- **Sloth mark:** Small, bottom-right or end card only; monochrome or single-color.  
- **No** cute sloth memes unless one ironic beat in a series later — this launch stays **premium calm**.  
- **Wordmark:** “STRATEGIC” heavy / “SLOTH” regular OR single lockup — match site.

### Cohesive account look
- **Covers:** Same template (hook typography + rule + mark).  
- **First frame:** Always readable at thumbnail size (test at 180×320).  
- **Series tag (optional):** Tiny “01/08” mono in corner for batch posting recognition.

# 2. Reusable reel template system

### Hook screen (0–1.5s)
- Full bleed `#0B0C0E`  
- One headline block, centered or lower-third  
- Optional 1-line sub-hook at 60% opacity  
- **Rule:** First 3 words must read in under 0.5s of attention  

### Body screen
- Same background family; alternate slightly (gradient angle ±15°) to avoid monotony  
- Max 3 lines; animate line-by-line or hard cut per line  
- Step reels: left-aligned number + right text block  

### CTA screen (last 1.5–3s)
- Simplified layout: one benefit line + **strategicsloth.com**  
- Secondary: “LINK IN BIO” in mono, smaller  
- Sloth mark appears here if nowhere else  

### Caption style (Instagram)
- Line break after hook thought  
- 2–3 short paragraphs  
- One clear CTA line with URL spelled once  
- 3–5 hashtags max (quality over spam); rotate per reel  

### Cover style
- **1080×1920** export with **safe center 1080×1400** for crop preview  
- Large 2-line hook + accent rule  
- Darken bottom 20% for title readability if you add small series text

# 4. Handoff for designer / self-serve (Canva · Figma · CapCut · Premiere)

**Creative direction summary**  
Strategic Sloth’s first reel batch should feel like **a quiet documentary confession with designer restraint** — not a hustle montage. Visuals stay dark, typography does the emotional work, and motion is **decisive but slow**. Every reel opens with a **thumbnail-readable hook** and closes on a **single CTA** to **strategicsloth.com** / link in bio.

**Canva**  
- Create brand kit: colors above + 2 fonts.  
- One “Hook / Body / CTA” page template; duplicate 8 times.  
- Export MP4 at 1080×1920; animate page transitions 0.2–0.4s.

**Figma**  
- Three components: `Frame_Hook`, `Frame_Body`, `Frame_CTA`.  
- Auto-layout text; export covers as PNG; use for storyboard PDF for editor.

**CapCut**  
- Save text styles as favorites after Reel 1; copy-paste across projects.  
- Use adjustment layer: slight contrast + desaturate greens if footage too lively.

**Premiere**  
- Essential Graphics master styles for Hook/Body/CTA.  
- Nested sequence per beat for easy ripple edits.

**Performance notes**  
- First 1 second = hook + visual tension.  
- Mid = one idea per card.  
- End = **one** CTA, no triple stack of links.  
- Optional A/B: talking head intro 2s → text body (often lifts saves on “story” reels).
